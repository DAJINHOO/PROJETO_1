# Academia Militar Imperial de Sinegard

## Sobre o Projeto

link do projeto: [https://github.com/DAJINHOO/PROJETO_1]

Este projeto foi desenvolvido para a disciplina de Desenvolvimento de Software como parte do Trabalho 02 – Estilização e Responsividade com CSS.

O objetivo do trabalho consiste na criação de um site institucional utilizando HTML5 e CSS3, aplicando conceitos de design responsivo, organização visual, acessibilidade e boas práticas de desenvolvimento web.

A instituição apresentada é fictícia e foi inspirada no universo da série de livros **A Guerra da Papoula (The Poppy War)**, da autora R. F. Kuang.

---

## Tema

A Academia Militar Imperial de Sinegard é uma instituição fictícia responsável pela formação de estrategistas, guerreiros, xamãs e líderes militares do Império de Nikan.

O site apresenta informações sobre:

- História da instituição
- Corpo docente
- Infraestrutura
- Cursos oferecidos
- Formulário de contato

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- Design Responsivo
- Menu Hambúrguer utilizando apenas HTML e CSS

## Responsividade

O site foi desenvolvido considerando três faixas de dispositivos:

| Dispositivo | Resolução |
|------------|------------|
| Mobile | até 767px |
| Tablet | 768px a 1023px |
| Desktop | 1024px ou superior |

O menu de navegação adapta-se para um menu hambúrguer em telas menores.
